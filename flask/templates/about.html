About this site

Hello world! This is a hobby project for one of my favorite online bullet hell multiplayer games RotMG. I've always wanted to learn more about the intersection of web development and data analysis, and this project was the perfect combination of the two. All the code is open-sourced on GitHub.

RealmCharts provides a live dashboard for insights on RotMG player count and Steam review analysis. Enjoy!

Attribution and Technologies

This website uses

  • Python, SQLite for data collection and the database
  • Flask and nginx for the webserver
  • HighCharts for interactive charting
  • JS/CSS for state management (no libraries!)

Also used are

  • RotMG pixel art for icons
  • Icomoon for GitHub icon
